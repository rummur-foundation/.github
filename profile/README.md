# Rummur Foundation

Open-source infrastructure for private communication.

We build [Rummur](https://github.com/rummur-foundation/rummur-protocol) — pronounced like *rumour* — a decentralized messaging protocol and family of clients. No servers. No accounts. No phone numbers. Messages travel as encrypted transactions on a live peer-to-peer network. Your keys are your identity.

The name is a palindrome: **R·U·M·M·U·R**. The same forwards and backwards — like the symmetric encryption protecting every message.

## Repos

| Repo | What it is |

| [`rummur-android`](https://github.com/rummur-foundation/rummur-android) | Android app — Kotlin + Jetpack Compose · GrapheneOS compatible |
| [`rummur-web`](https://github.com/rummur-foundation/rummur-web) | Browser extension + PWA (TypeScript + WASM) |
| [`rummur-device`](https://github.com/rummur-foundation/rummur-device) | Rummur Device — Linux OS image + open hardware design files |

## Get involved

The protocol spec is in active community review. Read
[`PROTOCOL.md`](https://github.com/rummur-foundation/rummur-protocol/blob/main/PROTOCOL.md)
and open a pull request if you have feedback — on the wire format, the
encryption scheme, or any assumption you'd challenge. Discussion happens
in the PR, not in private.
